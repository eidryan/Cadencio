Técnicas CSS

  • clip-path: Corta cantos criando formas angulares (chanfrado, diamante, paper-cut)
  • transform: skewX(): Inclina o botão mantendo texto legível
  • transform: rotate(): Leve rotação para efeito "solto"
  • ::before/::after: Camadas empilhadas, cantos dobrados com revelação de cor
